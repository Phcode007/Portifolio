## DESIGN SYSTEM

### Paleta de cores
```css
--green:      #3ddc84;   /* verde principal — accent */
--green-dim:  #2bbd6b;   /* verde hover */
--bg:         #050508;   /* fundo principal escuro */
--bg2:        #0d0d12;   /* fundo cards */
--bg3:        #13131a;   /* fundo card top */
--border:     rgba(255,255,255,0.07);
--text:       #e8e8f0;   /* texto primário */
--muted:      #6b6b80;   /* texto secundário */
--accent:     #7c6fff;   /* roxo para badge "Em Construção" */
```

### Tipografia
- **Display / Títulos:** `Syne` (Google Fonts) — peso 700/800, letra-spacing negativo (-0.03em a -0.04em)
- **Corpo / Código:** `DM Mono` (Google Fonts) — peso 300/400, letra-spacing sutil

### Tom visual
- Dark mode exclusivo
- Minimalismo técnico: parece a interface de um terminal sofisticado
- Grid de fundo sutil (linhas finas em rgba(255,255,255,0.07), tamanho 60×60px)
- Noise overlay leve (SVG feTurbulence, opacity 0.025) para textura
- Glow do cursor (radial-gradient verde que segue o mouse)

## COMPORTAMENTOS INTERATIVOS (JavaScript)

1. **Header com blur:** Ao scrollar >60px, adicionar classe `scrolled` que aplica `backdrop-filter: blur(16px)` e borda inferior
2. **Cursor glow:** Div `position: fixed` com `radial-gradient` verde que acompanha o mouse via `mousemove`
3. **Reveal animation:** `IntersectionObserver` detecta elementos com classe `.reveal` e adiciona `.visible` (opacity 0→1, translateY 28px→0) com delay escalonado entre irmãos
4. **Badge pulsante no Hero:** Um dot verde com animação `@keyframes pulse` (scale + opacity) ao lado do texto "Disponível para projetos"
5. **Hamburger menu mobile:** Toggle de nav em tela ≤768px com `transform: translateX`

## REGRAS DE QUALIDADE DO CÓDIGO

- **Semântico:** Usar `<header>`, `<nav>`, `<section>`, `<footer>`, `<article>` corretamente
- **Acessível:** `alt` em todas as imagens, `aria-label` nos botões ícone, contraste adequado
- **Performance:** Google Fonts com `preconnect`, imagens com `loading="lazy"` onde aplicável
- **CSS variables:** Todas as cores via `var(--nome)` no `:root`
- **Sem frameworks externos:** Somente HTML, CSS puro e JavaScript vanilla (sem jQuery, Bootstrap, etc.)
- **Animações via CSS:** Preferir `transition` e `@keyframes` para performance (GPU compositing)
- **Zero `!important`:** Estruturar seletores corretamente sem forçar especificidade
